# 获取bilibili up主的学历

提供python和js两种语言的代码。
## Python

```python 
pip install requests
```
```shell script
python get_bilibili_school.py
```

结果
```shell script
G:\Desktop\pythondemo\bilischool>python get_bilibili_school.py
该UP主的学校: 清华大学
```

## js
运行程序
```js
node get_bilibili_school.js
```
结果
```shell script
G:\Desktop\pythondemo\bilischool>node get_bilibili_school.js
该UP主的学校: 上海交通大学
```
